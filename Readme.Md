# 🐱🐶 Cat vs Dog Image Classifier using CNN

A Deep Learning based Image Classification project built using TensorFlow and Keras.

## Features

- Classifies images as Cat or Dog
- Convolutional Neural Network (CNN)
- Image preprocessing and normalization
- Model prediction system
- TensorFlow + Keras implementation

## Technologies Used

- Python
- TensorFlow
- Keras
- NumPy

## Project Structure

```
cat_dog_classifier.py
predict.py
dog.jpg
test.jpg
```

## Training

Run:

```bash
python cat_dog_classifier.py
```

## Prediction

Run:

```bash
python predict.py
```

## Note

The trained model file (.h5) is not uploaded because it exceeds GitHub's file size limit.

Users can train the model locally to generate the model file.
