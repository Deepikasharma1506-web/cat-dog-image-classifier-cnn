# 🐱🐶 Cat vs Dog Image Classifier

A Deep Learning based image classification web application that predicts whether an uploaded image is a Cat or a Dog.

## Features
- Upload image
- Real-time prediction
- Confidence score
- Deep Learning CNN model
- Deployed on Hugging Face Spaces

## Tech Stack
- Python
- TensorFlow / Keras
- NumPy
- Gradio
- Hugging Face Spaces
